<h1 align=center>Start project</h1>

```bash
# Run on terminal
git clone https://github.com/mfortunat0/Figma-clones

cd Figma-clones

# For yarn
yarn && yarn dev

# For npm
npm install && npm run dev
```

Acess <a href="http://localhost:3000/"> localhost:3000</a>
